JSON and AJAX

I created this JSON and AJAX app by adding an event listener to the Fetch Data button. Once clicked, the function sends a GET request to https://learnwebcode.github.io/json-example/animals-1.json and returns data back to the app. As there are multiple items in the json object, I used a for loop to list the information in seperate parragraph tags. this information is then concatinated with strings of text to create sentences.

After the Fetch button is pressed 3 times, a classList item of 'hide-me' is added to the button to hide it, and an id of 'btn-refreshed' is added to the refresh button. This then display the refresh button and allow the user to restart the app.
